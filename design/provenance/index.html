Mobile UI principles (Kole Jain)

Zakodowane na frame’ach M-01…M-09. Video: youtube.com/watch?v=Gfsd8NNuD9g

1. Navigation
4 primary tabs (Biblioteka, Osoby, Chat, Więcej) — ≤5. Deep routes hide tab bar; back affordance explicit (M-04, M-05, M-08).
2. Scale
Touch targets ≥44px (tab items, icon buttons, composer, CTA). Type 15–20px titles; comfortable phone density.
3. Content
Photo covers / faces / answer prose dominate first viewport. Chrome (tabs, rails) secondary to media and answers.
4. One screen, one job
List ≠ thread ≠ sources ≠ options. Secondary jobs in bottom sheets (M-06, M-07).
5. Gestures / patterns
Sheet handle + scrim (swipe-dismiss pattern), back from deep routes, native-feeling solid bottom tab bar (not floating glass), options via ⋮ sheet.
